# UwhLife Agent Notes

- 这是一个 Flutter 移动端项目，当前优先做首页骨架与高频校园服务入口。
- 视觉主色调改为绿色系，当前品牌主色为 `#22c55e`，整体方向参考微信绿，但要更浅、更轻、更适合现代移动端界面。
- 视觉风格保持简洁、留白充足，可以使用轻量绿色层次与柔和背景，不做厚重渐变和复杂装饰。
- 首页暂时只保留 4 个功能：门锁、付款码、洗澡/热水、智慧课堂。
- 底栏暂时只有 2 项：首页、头像。
- 设计时优先考虑 iPhone SE 3 等小屏设备，再兼容常见 Android 尺寸。
- 图标优先使用 Flutter 内置 `Icons` 的 `outlined` 或 `rounded` 风格，减少额外资源开销。
- 首页重绘时，顶部已有文案暂时保持不变；功能模块后续再拆分，当前阶段先不要改动功能行为。
